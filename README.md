# 🚀 DevHire AI Job Tracker

DevHire AI Job Tracker is a full-stack web application that helps job seekers organize and manage their job search efficiently. It provides a centralized platform to track applications, monitor progress, and organize opportunities through a clean and user-friendly interface.

## 🌐 Live Demo

https://devhire-job-tracker.netlify.app/

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 👤 User Registration & Login
- 📋 Track Job Applications
- ➕ Add, Edit & Delete Applications
- 📊 Application Status Management
- 🔍 Search and Filter Applications
- 📱 Responsive User Interface
- ⚡ RESTful API Integration
